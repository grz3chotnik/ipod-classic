# How to use

### install dependencies
> npm i
### replace CLIENT_ID and REDIRECT_URI
> Replace CLIENT_ID with sessionID from https://developer.spotify.com/ and REDIRECT_URI with your domain/ip
### run the client
> npm run dev



# How was it made?

- React + Typescript for the UI
- Spotify Web Playback SDK for spotify integration
- Scrollwheel works just like in the real iPod Classic
